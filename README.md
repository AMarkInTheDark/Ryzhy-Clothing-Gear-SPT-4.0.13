# Ryzhy Clothing & Gear - SPT 4.0.13

Backport of Ryzhy's clothing, facecover and bag for SPT 4.0.13.

Includes top, pants, balaclava and bag.

## Items

**Clothing (Ragman LL2 — requires completing "Setup" for Skier)**
- Ryzhy Top — 80,000₽
- Ryzhy Bottoms — 45,000₽

**Facecover (Ragman LL2)**
- Ryzhy Balaclava — 2,874₽
- Also spawns in duffle bags and on scav bodies

**Backpack (Ragman LL2)**
- Ryzhy Bag — 11,489₽

## Requirements

- SPT 4.0.13
- [WTT-CommonLib](https://forge.sp-tarkov.com/mod/2310/wtt-commonlib)

## How to install

1. Download the latest release `.zip`
2. Unpack it
3. Move the `RyzhyBackport` folder into your `SPT/user/mods/` directory
